Nugasin Class (Bootstrap)

Fitur di versi ini:
- Dark Mode / Light Mode (toggle di navbar, tersimpan di localStorage: `nc_theme`).
- Simulasi UAS (demo) + login VIP demo (`vip / vip123`).
- Paket Soal menggunakan JSON (sample format):
  - File: `assets/data/paket_agama_islam_sample.json`
  - Struktur: 3 sesi, masing-masing 50 soal (total 150).

Catatan:
- Login VIP masih demo (hardcode di frontend). Untuk produksi harus pakai auth dari backend (token/session).
- Paket soal saat ini baru ada 1 soal sample (Agama Islam) agar kamu bisa mengikuti format JSON-nya.
